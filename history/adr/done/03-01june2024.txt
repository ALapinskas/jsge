Add user feedback and questions / Добавить пользовательскую обратную связь и вопросы

Status: Done

Context: 
    Project doesn't have feedback form and possibility to ask question.
    Required to add such functionality, it should meet the following requirements:
    1. Simplicity: Easy to use.
    2. Global: Should be comfortable to use in any country.
    3. Two-languages: Russian/English

    Проект не имеет формы обратной связи и возможностей задать вопрос по использованию.
    Требуется добавить данный функционал, он должен отвечать следующим требованиям:
    1. Простота: максимально упростить сбор обратной связи и задавания вопросов.
    2. Глобальность: одинаково удобно использовать из любой страны, 
    3. Языки: английский/русский

    Варианты:
    - Телеграм канал: телеграм плохо развит в англоязычном сегменте, 
      легко поставить и поддерживать, это сторонний проект.
    - форум на discord: дискорд активно используется разработчиками игр, хорошая международная поддержка,
      легко поставить и поддерживать, это сторонний проект.
    - поставить php/mysql форумный движок: потребуется авторизация и аутентификация пользователей, 
      установка на свой сервер.

Decision:
    Create discord channel.
